# Operating System Images

This directory contains operating system images for the P2P nodes. Each node in the decentralized network functions as a complete operating system while participating in the P2P protocol.

## Image Types

- Base OS images with P2P node software pre-installed
- Container images for lightweight node deployment
- Virtual machine images for different platforms
